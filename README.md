# shukatsu-mail-guide
## 就活メール返信ガイド｜就活の木

AIを使った就活メール返信プロンプト生成PWAアプリです。

### 機能

- 8つのシーンからプルダウンで選択
  - 📩 受検案内への返信
  - 📅 面接日程の確認・調整
  - ✅ 内定承諾の返信
  - 🙏 選考辞退の連絡
  - ❓ 問い合わせ・質問
  - 🙇 お礼メール（面接後）
  - 📋 書類提出の連絡
  - 🔄 その他（自由入力）

- 先方からのメールを貼り付け
- 伝えたいことを箇条書き入力
- AIへの依頼プロンプトを自動生成
- ワンタップでコピー
- ChatGPT・Claude・Gemini・Copilotへの直接リンク
- PWA対応（ホーム画面に追加可能）

### 使い方

1. シーンをプルダウンで選ぶ
2. 届いたメールを貼り付ける
3. 伝えたいことを箇条書きにする
4. 「プロンプトを生成する」をタップ
5. コピーしてAIに貼り付ける
6. できたメールを自分で確認して送信
